该仓库包含多个基于 Java 开发的小型项目与工具类代码，覆盖数据抓取、模拟仿真、Web 开发、文本分析等领域，具体内容如下：
直播相关抓取工具：
CrawlDouyuDanmu：基于斗鱼开放协议的弹幕抓取代码，含 README 说明；
CrawlPandaDanmu：熊猫直播平台弹幕抓取程序，含 README 说明。
仿真模拟类项目：
InsulinPump：胰岛素泵仿真程序，模拟胰岛素泵工作逻辑；
MicroWave：微波炉模拟程序，支持两位数格式显示时间，已完成时间显示功能更新。
数据抓取工具：
PullBookinfo：豆瓣图书信息抓取工具，支持手动添加代理，适配新版本豆瓣接口；
medical_question：百度知道医学类问题及回答的抓取与存储工具；
com/crawl/comments：多线程评论抓取模块，采用 “每个线程独立使用 HttpClient 实例” 的设计，提升抓取效率；
DownloadImg.java：单文件工具，用于抓取并保存 Bing 首页图片；
Main.java：知乎模拟登录功能代码，已完善登录逻辑。
Web 开发与系统类：
Struts2FileUpAndDown：基于 Struts2 框架的文件上传与下载功能演示项目；
西邮导游系统源码：西安邮电大学校园导游系统代码，含校园相关地点介绍等功能。
文本分析类：文本情感分析：基础文本情感分类程序，提供完整的情感分析核心代码。
